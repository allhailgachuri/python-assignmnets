# 🐍 BDS 2205 – Python Programming Easy Assignments & CAT

## 📌 Overview

This repository contains solutions for **BDS 2205 – Python Programming** coursework, including:

* Assignment 1 (Python Basics)
* Assignment 3 (Modules, File Handling & Exception Handling)
* CAT 2 (Comprehensive Assessment)

The programs demonstrate fundamental to intermediate Python concepts through practical data processing tasks.

---

## 🎯 Objectives

The purpose of this repository is to showcase:

* Understanding of Python syntax and structure
* Application of core programming concepts
* Data handling and processing techniques
* Error handling and debugging
* Object-Oriented Programming (OOP) principles

---
